# 项目文件说明

## 一、配置文件

| 文件 | 作用 |
|------|------|
| [.env.example](.env.example) | 环境变量模板，定义 Ollama 地址、模型名称、向量库路径等配置项 |
| [.gitignore](.gitignore) | Git 忽略规则，排除 `__pycache__`、`.env`、`vector_store/` 等 |
| [requirements.txt](requirements.txt) | Python 依赖清单，`pip install -r requirements.txt` 安装全部依赖 |

---

## 二、config/ — 配置管理

| 文件 | 作用 |
|------|------|
| [config/\_\_init\_\_.py](config/__init__.py) | 包标识文件，使 `config` 成为可导入的 Python 包 |
| [config/settings.py](config/settings.py) | 加载 `.env` 配置，提供全局常量（`OLLAMA_BASE_URL`、`LOCAL_LLM_MODEL` 等）和工厂函数（`get_embedding_model()` 返回 Embedding 模型，`get_llm()` 返回 ChatOllama 实例） |

---

## 三、core/ — 核心业务逻辑

| 文件 | 作用 |
|------|------|
| [core/\_\_init\_\_.py](core/__init__.py) | 包标识文件 |
| [core/loader.py](core/loader.py) | 多格式文档加载器。`load_document(file_path)` 根据文件扩展名（PDF/TXT/MD）自动选择加载方式，返回 LangChain Document 列表 |
| [core/splitter.py](core/splitter.py) | 文本分割器。`split_documents(docs)` 使用 `RecursiveCharacterTextSplitter` 将长文档切分为 800 字符的块（overlap=100），过滤空块 |
| [core/vector_store.py](core/vector_store.py) | 向量数据库管理。`create_vector_store()` 构建 Chroma 向量库并持久化；`load_vector_store()` 加载已有库；`get_retriever()` 返回检索器 |
| [core/rag_chain.py](core/rag_chain.py) | RAG 问答链。`create_rag_chain()` 创建基于 `RetrievalQA` 的检索增强生成链；`query_rag(question)` 直接问答并返回答案+来源文档 |
| [core/tools.py](core/tools.py) | Agent 工具定义。用 `@tool` 装饰器定义三个工具：`get_weather(city)` 查询天气、`send_notification(msg, to)` 发送通知、`get_current_time()` 获取时间 |
| [core/agent.py](core/agent.py) | Agent 核心。`DigitalHumanAgent` 类实现意图分类 → 路由分发（tool/rag/chat），工具调用路径会解析 LLM 输出的 JSON 并实际执行工具函数 |
| [core/memory.py](core/memory.py) | 对话记忆管理。`SessionMemory` 类维护多轮对话历史，支持添加消息、获取上下文、裁剪超长历史、清空会话 |

---

## 四、voice/ — 语音处理

| 文件 | 作用 |
|------|------|
| [voice/\_\_init\_\_.py](voice/__init__.py) | 包标识文件 |
| [voice/stt.py](voice/stt.py) | 语音转文字。`transcribe(audio_bytes, model_size)` 使用本地 Whisper 模型将音频转为文本，支持切换模型大小，懒加载+缓存 |
| [voice/tts.py](voice/tts.py) | 文字转语音。`synthesize(text, lang)` 使用 gTTS 将文本转为 MP3 音频字节 |
| [voice/audio_utils.py](voice/audio_utils.py) | 音频工具函数。`pcm_to_wav()` 将 PCM 转 WAV、`audio_bytes_to_wav_file()` 保存临时文件、`cleanup_temp_file()` 清理临时文件 |

---

## 五、服务入口

| 文件 | 作用 |
|------|------|
| [server.py](server.py) | FastAPI 主程序。提供 `GET /` 状态检查、`POST /chat` 文本对话、`POST /reset` 重置会话、`WS /voice` WebSocket 语音交互端点。WebSocket 流程：接收音频 → STT → Agent 处理 → TTS → 返回音频 |

---

## 六、scripts/ — 辅助脚本

| 文件 | 作用 |
|------|------|
| [scripts/build_vector_store.py](scripts/build_vector_store.py) | 构建向量库。扫描 `documents/` 目录下所有 PDF/TXT/MD 文件，加载 → 分割 → 写入 Chroma 向量库 |
| [scripts/test_voice.py](scripts/test_voice.py) | WebSocket 测试客户端。支持文本对话测试和语音交互测试，可从文件加载音频或麦克风录音 |

---

## 七、数据目录

| 目录 | 作用 |
|------|------|
| [documents/](documents/) | 知识库源文件存放目录，用户自行放入 PDF/TXT/MD 文件 |
| [vector_store/](vector_store/) | 向量库持久化目录，由 `build_vector_store.py` 自动生成 |

---

## 八、文件依赖关系

```
server.py
  ├── config/settings.py        (读取配置)
  ├── core/agent.py             (对话处理)
  │     ├── config/settings.py  (获取 LLM)
  │     ├── core/tools.py       (工具定义)
  │     ├── core/memory.py      (会话记忆)
  │     └── core/rag_chain.py   (RAG 检索)
  │           └── core/vector_store.py (向量库)
  │                 └── config/settings.py (获取 Embedding)
  ├── voice/stt.py              (语音识别)
  └── voice/tts.py              (语音合成)

scripts/build_vector_store.py
  ├── config/settings.py
  ├── core/loader.py
  ├── core/splitter.py
  └── core/vector_store.py
```
